Inventory & billing

No REST endpoints are defined in this repo for inventory (8082) or billing (8083). They consume Kafka events. Per README, confirm behaviour with Kafka console consumer and service console logs after placing an order.

CORS is enabled on the gateway for browser calls from this static site. After changing gateway code, rebuild & redeploy the gateway image if needed.
